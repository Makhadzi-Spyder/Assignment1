# Time-Based Social Interaction Suggestion App

## Github Repository


## Description
This app suggestions various text messages and social interactions a person can have based on the time of the day

## Features
-User inputs time of day(morning, afternoon or night)
-App displays a suggestion
-The reset button clears input and output
-Toast messages for user error handling
